# Penguin Species Classification

## Overview

This project uses Machine Learning techniques to classify penguin species based on their physical characteristics.

## Dataset

The dataset contains information such as:

- Bill Length
- Bill Depth
- Flipper Length
- Body Mass
- Sex
- Island

Target Variable:

- Species

## Technologies Used

- Matplotlib
- Scikit-learn
